# Product Showcase

A simple product showcase website built with HTML and CSS.

## Features

* Responsive product cards
* Navigation menu
* Hover animations
* Clean and modern layout
* Product image gallery

## Technologies

* HTML5
* CSS3

## Project Structure

```text
site/
├── index.html
├── css/
│   └── style.css
└── img/
    ├── 1.jpg
    ├── 2.jpg
    └── 3.jpg
```

## Preview

This project displays a collection of products using card-based layouts with images, prices, and purchase buttons.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/product-showcase.git
```

Open `index.html` in your browser.
